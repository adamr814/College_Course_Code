Program #3

Write a program to create a class called Student:
• The class Student has two attributes: studentName, a String variable, and enrollNumber, an
integer variable.
• Assign the value of enrollNumber as 5 and that of studentName as John by creating an object of
the class Student.
• Assign and print the enrollNumber of two students having names Brian and John respectively by
creating two objects of class Student.
• The assignment of values to the variables should be done directly in the main method without
using any methods or functions.
